# Face Recognizer using FaceNet

A face recognition system built with FaceNet and OpenCV to capture, train, and recognize faces using deep learning embeddings.

## Features
- Capture face images from webcam  
- Train a model using FaceNet embeddings  
- Recognize faces in real time  
- Lightweight and simple  

## Tech Stack
- Python  
- OpenCV  
- FaceNet  
- NumPy  
- Scikit-learn  

## How It Works
1. Capture face images  
2. Extract embeddings using FaceNet  
3. Train the model  
4. Recognize faces in real time  

## Usage

### Capture Faces

python capture_faces.py


### Train Model

python train_model.py


### Recognize Faces

python recognize_face.py

- Attendance systems  
- Security systems
